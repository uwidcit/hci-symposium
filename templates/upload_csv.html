Upload CSV File
File must be in CSV format with headers matching the required columns

CSV Format Requirements

Required Columns
  • group_name: Research group name
  • title: Research project title
  • abstract: Project abstract/summary
  • member1_name: First team member
  • member2_name: Second team member
  • paper1_title: First paper title
  • paper2_title: Second paper title
Optional Columns
  • presentation_video_url: URL to presentation video
  • literature_review: Literature review content
Sample CSV Format
group_name,title,abstract,member1_name,member2_name,paper1_title,paper2_title,presentation_video_url,literature_review
"Team Alpha","User Interface Design for Mobile Applications","This research explores...","John Doe","Jane Smith","Mobile UI Design Patterns","User Experience Optimization","https://www.youtube.com/watch?v=example1","The literature review examines..."
"Team Beta","Accessibility in Web Design","Study of web accessibility...","Alice Johnson","Bob Wilson","Web Accessibility Implementation","Inclusive Design Principles","https://www.youtube.com/watch?v=example2","Current research shows..."
Tip: Use quotes around text fields that contain commas. The system will automatically handle special characters and line breaks.
Upload Process
  1. Prepare your CSV file with the required columns and data
  2. Upload the file using the form above
  3. Review the results - you'll see a success message with the number of projects uploaded
  4. Add media files (posters, videos) individually to each project as needed
  5. Manage projects from the admin dashboard
